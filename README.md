**Real-time ECTS-based GPA calculator for Bologna system students.**

## ⚡ Advanced Features

• 🧮 **Smart Exam Predictor** (calculates required final exam scores)

• 📋 **Constraint Validation** (prevents impossible grade combos) 

• 💾 **One-Click Export** (PDF-ready results) 

• 📱 **Fully Responsive** (mobile-first design) 

• 🌍 **RTL Support** (Arabic-ready)

• ✏️ **Editable Data** (ECTS & Titles)

• 🧮 **Accurate Calculation** (based on entered grades)

• 🎯 **Smart Grade Estimation** (closest valid grade combinations)
> Helps students understand achievable outcomes

> Useful for planning before exams
